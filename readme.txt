A widget that displays all relevant product categories for a given search result. Shortcode: [relevant_categories]

== Description ==

This is a very lightweight plugin that gives you a widget or shortcode which shows a list of all relevant product categories in a search query. It's kind of like the regular 'product categories list', but optimized for search result pages.

== Frequently Asked Questions ==

= How does it work? =

* Install the plugin
* Go to appearance --> Widgets
* Select 'Relevant Product Categories Widget'
* Fill in the title i.e.: "Relevant categories:"
* Make sure the sidebar where you placed the widget is enabled
* Search for a product! It should show the correct product categories.

= How can I use the shortcode? =

Pick the shortcode block in the block-editor and fill in this shortcode: [relevant_categories]
